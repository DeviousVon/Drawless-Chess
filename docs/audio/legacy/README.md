# Rejected imported audio evidence

`REJECTED_IMPORT_SHA256SUMS.txt` belongs to the original 2026-07-13 downloaded pack. It is
retained only to document what was audited and subsequently rejected during listening review.
It does not describe the current runtime library as a whole: 46 assets were rebuilt, while 58
source- and license-audited assets remain byte-identical carry-forwards.

The rejected import contained an el_boss slide layer in 22/50 moves and 8/12 captures, plus six
household pop/snap effects labeled as fireworks. The slide, can, camera, and cork sources were
removed from the active `docs/audio/source_recordings/` set. The MIT-licensed ion.sound
`snap.mp3` remains only for two reserved low-time cues; the current verifier forbids it and every
other non-Rudmer source from the firework pool.
